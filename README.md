# Python Weather App

A desktop application built with Python and Tkinter that displays current weather information for a given city using the OpenWeatherMap API.

## Features
- Fetches real-time weather data.
- Displays Temperature, Description, Humidity, and Wind speed.
- Interactive and user-friendly GUI.

## Prerequisites
- Python 3.x
- `requests` library

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aimahive7/py-weather-app.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Configuration
Before running the app, you need to provide your OpenWeatherMap API key:
1. Sign up at [OpenWeatherMap](https://openweathermap.org/) to get a free API key.
2. Open `weather_app.py` and replace `"YOUR_API_KEY_HERE"` with your actual key.

## Usage
Run the application:
```bash
python weather_app.py
```
